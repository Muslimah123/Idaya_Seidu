Enhancing Query Flexibility in the Malaria Data Access System

Authors: Idaya Seidu, Carine Pierrette Mukamakuza, Rose Mary Happiness, Pacome Simon Mbonimpa

Institution: Carnegie Mellon University, Kigali, Rwanda

Abstract: This report proposes an enhancement of the existing malaria data access model to support user-defined queries, improve data security, and optimize resource usage. The anticipated model aims to provide a robust framework for healthcare professionals and policymakers to engage with data more interactively and effectively, thereby improving decision-making in malaria control strategies.

Keywords: Malaria surveillance, data access model, query flexibility, user-defined queries, data security, resource optimization, decision-making
